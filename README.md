## 📦 01 · Optimizing Retail Operations

> **5-page Power BI dashboard analyzing 500K+ retail transactions**

Built an interactive Power BI report covering sales performance, customer segmentation, product profitability, and time-series trends. Integrated 5 datasets (Sales, Products, Stores, Customers, Dates) in a star-schema model with full DAX KPI calculations.

**Highlights:**
- Total Revenue: 230.47M · Profit Margin: 24.7% · Transactions: 500K
- Regional revenue breakdown, payment mode analysis, discount impact
- Customer loyalty segmentation and age-group revenue comparison

## 📱 02 · Influencer Marketing Effectiveness

> **Power BI dashboard evaluating ROI and customer value across influencer campaigns**

Analyzed a fashion brand's influencer marketing program using Power BI and statistical ANOVA testing. Evaluated ROI, revenue vs cost, customer lifetime value (LTV), and average order value (AOV) across influencers, campaigns, and channels.

**Highlights:**
- Total Revenue: 352.85M · Avg ROI: 296₹
- BeautyBuzz and SustainableChic identified as top-performing influencers
- Instagram delivers highest AOV; all ANOVA tests significant (p ≈ 0)

[![View Project](https://img.shields.io/badge/View%20Project-project--2-F2C811?style=for-the-badge&logo=powerbi&logoColor=black)](../../tree/project-2)

## 🛍️ 03 · Retail Growth: Sales & Consumer Behavior

> **1M+ transaction analysis using Python statistical testing + Power BI dashboards**

Applied Two-Way ANOVA, t-tests, RFM segmentation, and seasonality decomposition on a large retail dataset. Built a 2-page Power BI dashboard to visualize revenue drivers and customer retention strategy.

**Highlights:**
- Total Revenue: 1.88bn across 1M+ customers
- Moderate discounts outperform heavy discounting — statistically validated
- RFM segmentation identifies Champions, Loyal, At-Risk, and New segments

## 🚛 04 · Logistics Intelligence Dashboard

> **5-page Power BI report integrating 14 logistics datasets with advanced DAX**

Built a comprehensive logistics intelligence report covering fleet utilization, driver safety, fuel cost optimization, and financial profitability. Designed a star-schema relational model and implemented advanced DAX measures including safety scoring, utilization rate, and profit per mile.

**Highlights:**
- Total Revenue: 298.62M · Gross Profit: 197.30M · Profit per Mile: ₹1.62
- Fleet Utilization: 83% · Fuel Cost: 95.59M (largest expense driver)
- DOT violations identified as primary cause of delivery delays
